## Installation

#### Clone the project

```bash
  git clone https://github.com/AliNikandish/BudgetApp.git
```

#### Go to the project directory

```bash
  cd BudgetApp
```

#### Install BudgetApp with npm

```bash
  npm install
```
#### Run:
```bash
  npm run dev
```
